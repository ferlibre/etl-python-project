# Projeto ETL com Python

Este é um projeto simples de ETL (Extração, Transformação e Carga) feito em Python.

## O que o projeto faz

- Lê dados de um arquivo CSV (clientes.csv)
- Gera mensagens personalizadas para cada cliente
- Salva o resultado em um arquivo (saida.txt)

## Tecnologias utilizadas

- Python
- CSV

## Como executar

1. Abra a pasta do projeto
2. Execute no terminal:

python main.py

## Objetivo

Praticar conceitos básicos de manipulação de dados e fluxo ETL.
